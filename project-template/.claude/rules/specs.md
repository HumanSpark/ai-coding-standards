# Spec documents

All feature specs, design documents, and planning outputs go in docs/plans/.
This applies regardless of which tool or workflow produces them - project-intake
skill, Superpowers brainstorm, manual creation, or any other flow.

Never create spec directories elsewhere (e.g. docs/superpowers/specs/,
docs/specs/, docs/design/).

The spec template is at docs/SPEC-TEMPLATE.md. Use it for all new specs.
